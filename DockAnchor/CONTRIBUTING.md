# Contributing to DockAnchor

Contributions are welcome! If you have ideas, bug reports, or improvements, please open an issue or submit a pull request.

## Getting Started

1. Fork the repository.
2. Create a feature branch.
3. Make your changes and ensure the project compiles (`swift build`).
4. Submit a pull request.

## Reporting Issues

Please use the [GitHub Issues](../../issues) page to report bugs or request features.
